# Contador de Cliques

Projeto simples feito em React que mostra um contador na tela e incrementa o valor a cada clique no botão.

## O que o projeto faz

- Exibe um número na tela (contador)
- Possui um botão
- Cada clique aumenta o contador em +1

## Tecnologias usadas

- React
- JavaScript
- CSS

## Como executar

```bash
npm install
npm start
